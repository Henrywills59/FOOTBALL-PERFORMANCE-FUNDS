# Phase 1 Scaffold

This phase establishes a clean production-ready foundation without domain features.

## Scope

- React + Vite frontend placeholder
- Express backend with `/health`
- Shared TypeScript package
- Prisma schema package
- PostgreSQL Docker service
- Environment templates
- Root build command

## Deferred

- Payments
- Football data integrations
- AI workflows
- Investor workflows

## Deployment Notes

The frontend is ready for Vercel as a Vite application. The backend can be deployed as a Node service, with environment variables supplied by the deployment platform.
